FGFXLargeScalePerceptualObscuranceIrradiance Plugin for UEVR
============================================================

Original shader:
  FGFXLargeScalePerceptualObscuranceIrradiance - Large Scale Perceptual Obscurance and Irradiance
  Version: 0.7.1 [ReShade 3.0]
  Source: https://github.com/AlexTuduran/FGFX/blob/main/Shaders/FGFXLargeScalePerceptualObscuranceIrradiance.fx
  License: MIT

Default source configuration preserved in this port:
  LSPOIRR_AUTO_GAIN_ENABLED = 1
  LSPOIRR_AUTO_GAIN_SPEED = 0.04
  LSPOIRR_BLUR_MAX_RECIPROCAL_THRESHOLD = 0.05
  LSPOIRR_CASCADE_3_ON = 0
  LSPOIRR_SRGB = 0
  ___CUSTOM_OCCLUSSION_IRRADIANCE_NEUTRAL_POINT = 0

Modifications: Ported from ReShade FX to DX11/DX12 C++ plugin with
embedded HLSL pixel shaders.

The UEVR plugin wrapper code is distributed under this repository's license.
